# Ambiente de Desenvolvimento Java 17 + Angular 20

Este projeto utiliza **Docker** e **VS Code Dev Containers** para fornecer um ambiente completo e padronizado.

## Tecnologias

- Java 17 (OpenJDK)
- Node.js 20
- Angular CLI
- Ubuntu Bookworm

## Como usar

1. Certifique-se de ter o Docker e a extensão "Dev Containers" no VS Code.
2. Clone este repositório.
3. Abra no VS Code e clique em "Reopen in Container".
